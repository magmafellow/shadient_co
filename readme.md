# Shadientco layout

## usage for shared server
Use shadientco folder (duplicate of prepared_for_shared_site) for static demonstration on remote server (VPS)

## Idea
Supposed that I have only one server and want to present some projects. All 
projects located in according folder in root. e.g. /my-site/shadientco; /my-site/travel-agency

In all projects folders pathes are **aligned** so we can handle routing right. It is the heavy part of this idea,
because all projects located not directly in root but in root + project_folder
